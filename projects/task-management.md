# Task Management SaaS

A full-stack task management application designed to explore modern backend development concepts, authentication, authorization, collaboration features, and cloud deployment practices.

## Project Links

- Live Demo:
- Github Repository: [https://github.com/floatingnahid22/task-management](https://github.com/floatingnahid22/task-management)

---

## Features

### Authentication

- Register
- Login
- Refresh Token
- Forgot Password
- Email Verification

### Organizations

- Create Workspace
- Invite Members

### Tasks

- Create Task
- Update Task
- Delete Task
- Assign Task
- Due Dates
- Labels
- Comments

### Roles

- Owner
- Admin
- Member

### Frontend

- Dashboard
- Boards
- Task Modal
- Authentication Pages
